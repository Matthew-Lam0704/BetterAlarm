# SilentAlarm

A modern 4-page alarm app with a clean UI. Built with vanilla HTML, Tailwind CSS, and JavaScript.

## Pages
- **Alarms** – Manage alarms, toggle on/off, add new alarms
- **Sleep** – Bedtime schedule and wind-down settings
- **Stats** – Sleep overview and alarm usage
- **Settings** – Dark mode, notifications, default sounds

## Run locally
```bash
npm start
# or: python3 -m http.server 3000
```
Then open **http://localhost:3000** in your browser.
